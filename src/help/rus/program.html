Настройка — Программа
Рассмотрим по порядку опции самой программы FreeCap.
Только один экземпляр программыозначает что при повторной попытке запуска, FreeCap отобразит своё главное окно, если оно оказалось скрытым или свернутым
Запуск вместе с системойзапуск FreeCap вместе с Windows
Запускать свернутым в трей
Сворачивать в трей		при запуске FreeCap автоматически сворачивается в трей
Добавлять "via FreeCap" к заголовкам программДобавлять строчку "via FreeCap" к заголовкам соксифицируемых приложений. Удобно когда нужно найти скажем окно соксифицированного Internet Explorer-а среди других его окон
Показывать предупреждающие сообщенияесли не отмечено, сообщения пишутся в лог
Использовать трюки для проброса UDPЗаставляет FreeCap использовать нулевые src-порты для дейтаграмм. Работает на модифицированных SOCKS серверах для проброса через NAT
Прятать FreeCap по Alt+F4Прятать окно в трей, вместо закрытия приложения. Работает если поставлена галочка "Сворачивать в трей"
Разрешение имен DNS
ЛокальноОпределять IP адрес в соотвествии с настройками DNS вашего сетевого подключения.
Локально потом через проксиПопытатся определить адрес сначала локально, а если не получится то через прокси сервер
Только через проксиDNS сервер отсутсвует, определять все адреса только через прокси.
