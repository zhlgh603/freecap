Konfiguration — Program
Hier sind die Optionen von FreeCap:
Only one program instanceVerhindert irrtümliches starten mehrerer Instanzen von FreeCap.
Run at system startupAutostart FreeCap beim hochfahren
Run minimized to system tray,
Minimiere in die Systemleiste		Nach dem starten verschwindet FreeCap in die Systemleiste
Add "via FreeCap" to captionsZeigt "via FreeCap" in der oberen Namensleiste des Programmfensters das über FreeCap läuft. Damit ist leicht zu erkennen welches Broweserfenster über den Proxy läuft.
Show warnings in dialog boxesUnterdrückt Fenster mit Warnungen, diese werden stattdessen in die Log Datei geschrieben.
Use some tricks to traverse UDPFreeCap benutzt "zero src-ports" für einige modifizierte SOCKS server. Damit kann UDP eventuell über NAT laufen.
Hide FreeCap by Alt+F4Funktioniert nur wenn "Minimize to system tray" angehakt ist.
DNS Auflösung
LocalFreeCap benutzt den DNS Server der bei Ihren Netzwerkeinstellungen angegeben ist.
Local then remoteFreeCap versucht zuerst die DNS lokal zu finden, wenn das nicht klappt wird es über den Proxy versucht.
RemoteAlle Adressen werden über den Proxy aufgelöst.
