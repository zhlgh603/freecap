FreeCap — configuration
All configuration procedure contain from two steps -- Adding new application that should be socksified and proxy configuration
When configuration will be done, you can run SOCKSified program
By double click in the FreeCap main window
Or from the context menu
Your program itself doesn't require any additional configuration. FreeCap will take care that your program won't ever suppose that it's working through proxy
