FreeCap
The FreeCap program was created as alternative to "freeware" Permeo's SocksCap and designed to work with programs which doesn't have native proxy support. The main features of FreeCap is:
  • Supporting as SOCKS proxy as HTTP(S) proxy
  • Supporting proxy chaining of several proxy types (for example SOCKSv4 -> HTTP -> SOCKSv4 -> SOCKSv5)
  • Fully RFC compilant 1928, 1929, 2817 (SOCKS v4 è v5, authorization for SOCKSv5 and HTTP CONNECT)
  • DNS-resoving via any proxy
  • Step-by-step DNS resolving trought proxy chain
This and many other features makes FreeCap very useful tool.
